# OpenRevi

## Overview

OpenRevi is a platform designed to streamline and enhance the collection of authentic product reviews. By connecting businesses with engaged users, OpenRevi enables seamless distribution of products directly to consumers' email inboxes for genuine and insightful feedback.

[! Openrevi](https://share.gifyoutube.com/KzB6Gb.gif)](https://youtu.be/_7KR9Qo-kHA)



## Features

- **Product Distribution**: Businesses send products to selected users via email.
- **User Engagement**: Users test products and provide detailed reviews.
- **Review Submission**: Comprehensive feedback including text, ratings, images, and videos.
- **Insights and Analysis**: Businesses access user-generated content and analytics for product improvement.

## Benefits

### For Businesses
- **Authentic Feedback**: Honest reviews from real users.
- **Enhanced Visibility**: Boost brand credibility and visibility.
- **Product Improvement**: Utilize feedback to refine and improve products.

### For Users
- **Early Access**: Exclusive access to new products.
- **Influence**: Impact product development with your opinions.
- **Rewards**: Enjoy perks for participating in reviews.

## Getting Started

### Prerequisites

- Node.js and npm installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/openrevi.git
   ```
2. Navigate to the project directory:
   ```bash
   cd openrevi
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Configuration

Create a `.env` file in the root directory and add your environment variables:

```
REACT_APP_PUBLIC_POSTHOG_KEY=your_posthog_key
REACT_APP_PUBLIC_POSTHOG_HOST=https://us.i.posthog.com
```

### Running the Application

Start the development server:

```bash
npm start
```

The application will be available at `http://localhost:3000`.

## Usage

1. **Businesses**: Partner with OpenRevi to distribute your products for reviews.
2. **Users**: Receive products, test them, and submit your reviews through the platform.
